# Fullstack Authorization System 🔐

Полноценная современная система авторизации и управления профилем с поддержкой двухфакторной аутентификации (2FA) и входа через сторонние сервисы (OAuth). Проект разделен на независимый бэкенд на NestJS и клиентское приложение на Next.js (App Router).

## 🚀 Основной функционал

- **Регистрация и Вход**: Аутентификация на основе JWT-токенов (Access / Refresh) с безопасным хранением в куках.
- **Кастомный Google OAuth**: Авторизация и регистрация через учетную запись Google, реализованная вручную без использования сторонних библиотек-стратегий.
- **Двухфакторная аутентификация (2FA)**: Возможность включения/выключения двухфакторной защиты в настройках профиля.
- **Email-уведомления**: Красивые транзакционные письма для подтверждения аккаунта и кодов 2FA (интегрирован SMTP от Beget).
- **Управление профилем**: Интерактивная форма изменения личных данных (Имя, Почта) с полной валидацией.
- **Сессии и Кэширование**: Оптимизация работы и безопасное хранение временных данных/сессий в Redis.
- **Ролевая модель**: Разграничение прав доступа для пользователей (`REGULAR`, `ADMIN`).

## 🛠️ Стек технологий

### Frontend (Клиент)
- **Next.js 14+ (App Router)** — React-фреймворк с поддержкой серверных и клиентских компонентов.
- **TypeScript** — строгая типизация для предсказуемой разработки.
- **TanStack Query (React Query)** — эффективное управление серверным состоянием, кэшированием и мутациями.
- **React Hook Form & Zod** — обработка форм, сбор данных и клиентская валидация по схемам.
- **Shadcn UI & Tailwind CSS** — современный, доступный и адаптивный интерфейс компонентов.
- **Sonner** — всплывающие тост-уведомления для отображения статуса операций.

### Backend (Сервер)
- **NestJS** — прогрессивный Node.js фреймворк для создания надежных и масштабируемых приложений.
- **Prisma ORM** — современный инструмент для работы с базой данных PostgreSQL.
- **Docker & Docker Compose** — контейнеризация инфраструктуры (БД и Кэш).
- **PostgreSQL** — надежная реляционная база данных.
- **Redis** — хранилище данных в оперативной памяти для кэширования сессий и токенов.
- **Ручная JWT & OAuth авторизация** — собственная реализация механизмов аутентификации и работы с токенами без использования Passport.js.
- **Argon2** — современный и безопасный алгоритм хэширования паролей.
- **@nestjs-modules/mailer** — модуль интеграции почты для NestJS, работающий через SMTP Beget.
- **React Email** — современная и удобная вёрстка компонентов писем на React.

## 📦 Структура проекта

Проект организован в виде монорепозитория (две независимые папки):
- `nestjs-server/` — Серверная часть приложения (API, База данных, Авторизация, Шаблоны писем).
- `nextjs-app/` — Клиентский интерфейс (Страницы, Формы, Компоненты).

## 🔧 Локальный запуск

### 1. Клонирование репозитория
```bash
git clone https://github.com
cd fullstack-authorization
```

### 2. Настройка и запуск бэкенда (`nestjs-server`)
1. Перейдите в папку сервера:
   ```bash
   cd nestjs-server
   ```
2. Установите зависимости через Yarn:
   ```bash
   yarn install
   ```
...
7. Запустите сервер в режиме разработки:
   ```bash
   yarn start:dev
   ```

### 3. Настройка и запуск фронтенда (`nextjs-app`)
1. Откройте новое окно терминала и перейдите в папку клиента:
   ```bash
   cd nextjs-app
   ```
2. Установите зависимости:
   ```bash
   yarn install
   ```
...
4. Запустите приложение в режиме разработки:
   ```bash
   yarn dev
   ```
